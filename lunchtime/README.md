# 점심시간

`baksu` 루트 사이트 안에 들어가는 점심시간 익명 소통 프로토타입입니다.

## 포함된 흐름

- 하위 서비스 홈: `(주) 백수` 루트 홈으로 돌아가는 동선과 점심시간 입장 동선
- 스프레드시트형 첫 화면: 열/행 헤더, 비상호작용 셀 배경, 병합 셀 영역
- 상단 배너: 오른쪽에서 왼쪽으로 흐르는 정보 티커
- 셀 배경: 실제 DOM 셀로 구성하되 선택/하이라이트 없이 배경처럼 표시
- 시간 기준 전환: 24h / 12h
- 채팅 보내기: 중앙 하단 버튼에서 미니 채팅창 열기

## 실행

`baksu` 루트에서 정적 서버를 열면 `/lunchtime/`로 접속할 수 있습니다.

```bash
python3 -m http.server 8766 --bind 127.0.0.1
```

파일만 열어도 동작하지만, 루트 홈과 상대 링크를 함께 확인하려면 서버 방식이 더 안전합니다.
